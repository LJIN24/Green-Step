 # GreenStep

채식한 날을 기록하면 절감한 탄소를 ‘발걸음’으로 보여주며, 채식을 통한 친환경 챌린지에 쉽게 참여할 수 있는 앱입니다.
## ⚒️ Tech Stack

- **Framework**
  - UIkit
    
- **Architecture**
  - MVVM
  
- **Library**
  - RealmSwift

    * 로컬 저장을 위해서 사용했습니다.
    * 비교적 간단한 코드로 구현할 수 있고 CoreData보다 속도가 빨라서 사용했습니다.

- **Dependency Management**
  - Swift Package Manager
